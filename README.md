# ta-tetris-game

> Web Tetris Game


## 开发

使用 tmux-agent (ta) 进行任务管理：

```bash
# 创建新任务
ta spawn feat-xxx --agent codex --prompt-text "任务描述"

# 查看任务状态
ta task-status

# 查看任务日志
ta logs feat-xxx

# 合并 PR
ta merge feat-xxx
```
